# Firebase Crawler

A Python utility to search and query Firebase Firestore collections globally.

## Features
- Search across multiple Firestore collections
- Case-insensitive string matching
- Retrieve documents and fields containing search terms

## Setup
1. Download your Firebase service account key
2. Update the certificate path in the script
3. Install dependencies: `pip install firebase-admin`

## Usage
```bash
python firebase_search.py
```

Edit `search_term` variable to search for different values.
